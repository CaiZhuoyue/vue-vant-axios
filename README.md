# vue-vant-axios

## 技术栈
* vue
* vue-router 路由
* vant ui库
* axios ajax库
* scss css预处理器

## 项目文件架构说明

```js
|vue-vant-axios
    |——src 源码文件
      |——assets 公共资源
      |——components 公共组件
      |——router 路由
      |——views 逻辑页面
          |——home 首页
            |——index.vue 模板
            |——styles.scss 样式
            |——images 图片
          |——mime 主页
            |——index.vue
            |——style.scss
            |——images

```
